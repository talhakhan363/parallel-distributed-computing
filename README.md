This repository contains study materials, lecture notes, and lab implementations for the Parallel and Distributed Computing course.

The course transitions from low-level system calls to high-level parallel APIs:
* **Operating Systems:** Linux/UNIX environments and Shell scripting.
* **Shared Memory Programming:** Multithreading with POSIX threads and **OpenMP**.
* **Distributed Memory Programming:** Cluster computing using **MPI** (Message Passing Interface).
* **Heterogeneous Computing:** General-Purpose GPU programming with **NVIDIA CUDA (C/Python)**.
